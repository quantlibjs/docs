Class HullWhite

Single-factor Hull-White (extended Vasicek) model class.

This class implements the standard single-factor Hull-White model defined by \f[ dr_t = (\theta(t) - \alpha r_t)dt + \sigma dW_t \f] where \f$ \alpha \f$ and \f$ \sigma \f$ are constants.

test calibration results are tested against cached values

bug When the term structure is relinked, the r0 parameter of the underlying Vasicek model is not updated.
