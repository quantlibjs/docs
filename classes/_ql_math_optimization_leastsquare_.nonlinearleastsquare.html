Class NonLinearLeastSquare

Non-linear least-square method.

Using a given optimization algorithm (default is conjugate gradient),

\f[ min { r(x) : x in R^n } \f]

where \f$ r(x) = |f(x)|^2 \f$ is the Euclidean norm of \f$ f(x) \f$ for some vector-valued function \f$ f \f$ from \f$ R^n \f$ to \f$ R^m \f$, \f[ f = (f_1, ..., f_m) \f] with \f$ f_i(x) = b_i - \phi(x,t_i) \f$ where \f$ b \f$ is the vector of target data and \f$ phi \f$ is a scalar function.

Assuming the differentiability of \f$ f \f$, the gradient of \f$ r \f$ is defined by \f[ grad r(x) = f'(x)^t.f(x) \f]
